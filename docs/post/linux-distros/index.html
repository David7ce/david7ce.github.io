Intro#

Before diving into Linux, you need to decide which Linux distribution (distro) and desktop environment (DE) you want. A distro is an OS that bundles the Linux kernel with other software, making it easier to install and use. In this article, we’ll cover Linux distros and desktop environments.

Linux Distros#

Distros differ mainly in their package managers and intended use cases (personal computers, servers, smartphones, etc.)

Linux distributions are generally very similar, with the main differences being the package manager and the default desktop environment they ship with. However, the real distinction lies in the package manager.

A Linux distro typically includes:

  • Linux Kernel: The core of the OS.
  • GNU tools and libraries: Hence the name GNU/Linux.
  • Bootloader: Software that loads the OS, e.g., GRUB, systemd-boot.
  • File system: Manages how data is stored, e.g., EXT4.
  • Init system: Manages system initialization, e.g., systemd.
  • Drivers: Ensure hardware components function correctly.
  • Package Manager: Software that manages app installation, upgrades, and removal, e.g., pacman for Arch Linux.
  • Desktop Environments: Provide a graphical user interface (GUI) for ease of use. Popular DEs include KDE, GNOME, XFCE, and Mate.
    • Window System: Manages the display, e.g., Xorg (X11) or Wayland.
    • Window Manager: Controls window behavior, e.g., floating (KDE Plasma) or tiling (i3 WM).
    • Login Manager: Manages user logins, e.g., LightDM, SDDM.
    • Menus and Launchers: Like dmenu and rofi.
    • Global Themes: Customize the look and feel, including taskbars and wallpapers.
    • GUI Apps: Include login manager, bar manager, launcher, file manager, terminal, web browser, text editor, etc.

What I Recommend#

First choose the desktop environment that suit your preferences and then choose the base distro that most satisfy your needs. Doing this you will not waste your time switching from one DE to another frequently, concept know as distro-hopping which many Linux “streamers” do nowadays for content.

If you want to explore other options consider do it on a virtual machine and change your workflow if it is better.

Easy options#

If you’re looking for a family-friendly option, consider using a major Linux distro with a preinstalled desktop environment (DE) like GNOME or KDE. These are based on well-known base Linux distros such as Arch, Debian, or Fedora.

  • Arch-based:
    • EndeavourOS with KDE.
    • Garuda Linux, CachyOS, ArcoLinux with multiple options by default using KDE.
  • Debian-based:
    • Ubuntu: with GNOME or Kubuntu with KDE.
    • Linux Mint with Cinnamon, Mate, XFCE.
    • KDE neon with KDE.
    • Kali Linux with XFCE plus extra packages for hacking.
    • Proxmox with Openbox.
  • Fedora-based:
    • Fedora by default with GNOME + different variants for DEs as Fedora Spins.
    • Nobara with customized KDE.
    • Ultramarine with GNOME.

More Challenging Paths:#

Avoid this as a regular installation, this is great for learning the first time.

For those who want to delve deeper and understand the installation process, consider these distros doing manual installation via terminal.

  • Arch Linux: Manual installation, offering maximum customization.
  • Gentoo: Source-based distro, known for its flexibility and performance.
  • Void Linux: Independent distro with its own package manager (XBPS).
  • Slackware: One of the oldest distros, focuses on simplicity and stability.

Installation Modes:#

Here’s a list of installation methods to install a Linux distro from easiest to hardest:

  1. 🐧🦑 Penguin-Calamares Mode: Install a preconfigured distro (Linux kernel + DE + Calamares installer).
  2. 🐙 Octopus Mode: Automate the installation like an octopus using a bash script.
  3. 🐍 Python Mode (only with Arch): Use a Python script, like archinstall, to interact with the shell and select options through a form.
  4. 🐢 Turtle Mode: Build from scratch on the shell, enter directly the text terminal of the OS and install manually typing commands.

By following this guide, you can choose the right Linux distro and installation method based on your needs and experience level.
