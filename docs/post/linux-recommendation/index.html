Linux distros that I recommend#

To begin let me tell you that “all Linux distros” are very similar, the main differences are the package manager and the custom desktop environment that ship (window manager, file manager, editor, viewer, web browsers, etc). But the real difference between distros is the package manager.

There are around 600 distros maintained however based on Distrowatch there are only 44 independent distros and around 10 big Linux base distros.

In the top 10 Linux distros we find some community maintained distros such as Alpine, Arch, Debian, Gentoo, NixOS, OpenSUSE, Slackware, Void and others maintained by companies such as Ubuntu, Fedora, OpenSUSE.

Typically, Linux distributions are bundled with desktop environments such as GNOME, KDE, XFCE, or a combination of Wayland compositors and tools like Sway/Waybar. Additionally, some distributions offer variations known as flavors or spins, which are essentially the same distribution but with different preconfigured desktop environments. For example, Ubuntu utilizes GNOME, while Kubuntu employs KDE. Similarly, Fedora’s standard edition features GNOME, but there are also Fedora variants like Fedora KDE, Fedora XFCE, Fedora Cinnamon, and so forth.

However, certain distributions, such as Arch, Debian and Gentoo, do not come preloaded with a desktop environment. Instead, users are presented with a terminal interface upon installation, allowing for maximum customization and flexibility in selecting and configuring their preferred desktop environment or window manager.

Here are some example of Linux derivatives distros or based Linux distros:

  • Arch-based: EndeavourOS, Garuda Linux, CachyOS, ArcoLinux
  • Debian-based: Ubuntu, Kubuntu, Linux Mint, KDE neon, Kali Linux, Proxmox
  • Fedora-based: CentOS, Nobara, Rocky Linux, Ultramarine

Installation mode#

I have defined these method to install Linux from easy to hard as these:

  • 🐧🦑 Penguin-calamar mode: Base Linux distro with calamares installer
  • 🐙 Octopus mode: bash script installation
  • 🐍 Python mode (only with Arch): archinstall using python
  • 🐢 Turtle mode: build from scratch on the shell
