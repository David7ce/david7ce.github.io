Easy encryption

Encrypt your files in an easy way
2023-11-15 · 1 min · 114 words · Me

Easy encryption#

To do that create a ods file with LibreOffice Calc and “Save as” and select “Save with password”. You can also encrypt with GPG key.

  • Pros.
    • Don’t expose your data opening in web browser (like Bitwarden or vaultwaredn) with JS malfunctions.
    • One file in local
    • You can put only initial and end of password, is plain text
    • You can organize in you own way.
    • columns: email, category, company, username, password, notes
    • rows: as the values.
    • Just copy, paste or overwrite between files
  • Cons.
    • There is no option to autogenerate password
    • The file can be cracked with multiple tries, and depends on the strength password.
    • No option to hide password
    • No real synchronization
